static u8_t MTIMER_vAdjustDutyCycle ( u8_t  u8Channel,
u8_t  u8Mode,
u8_t  u8DutyCycle 
)
static

This function is used to adjust the duty cycle of the PWM modes whether the PWM mode is inverting or non-inverting.

Parameters
[in]u8ChannelThe Timer channel to be used
See also
Timer Channels
Parameters
[in]u8ModeThe Timer mode to be used
See also
Timer Modes
Parameters
[in]u8DutyCycleThe duty cycle to be used for the PWM modes
Note
This function should be used only in PWM modes (Fast PWM, Phase Correct PWM)
